# LAS Crashpad — Design Reference

## Reference: Existing Design (Ground Truth)

This project is a **reconstruction** of an existing design from a provided screenshot. The reference image is the ground-truth spec. Fidelity to the reference OVERRIDES all other design guidance.

### Confirmed Design System from Screenshot

**Design Movement:** Desert Electric / Neo-Vegas Brutalism
- Near-black background (#0D0D0D / ~#111)
- Asymmetric split hero: left panel = neon sign logo image with warm amber glow border; right panel = headline + CTA content
- Nav: dark bar, small logo mark + "THE LAS CRASHPAD" wordmark left, "BOOK NOW" CTA button right in hot magenta/orange

**Color Palette:**
- Neon orange: #FF6B1A (primary CTA, subheadline, stats)
- Electric blue: #3BB5FF (logo accent, bullet markers)
- Hot magenta: #FF2D78 (secondary accents)
- Warm white / cream: #F5EDD6 (main headline, body copy)
- Near-black: #0D0D0D (background)

**Typography:**
- Bebas Neue — display/headlines (BEST VALUE IN VEGAS, ONLY MINUTES TO TERMINAL)
- DM Sans — body copy, labels, stats

**Layout:**
- Full-viewport hero split: left ~45% (logo panel), right ~55% (content)
- Left panel: neon sign image centered with amber/orange glow box shadow
- Right panel: large Bebas Neue headline, orange subheadline with star burst icon, body copy, CTA buttons row, trust bar stats
